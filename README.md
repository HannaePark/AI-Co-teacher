## 📝 프로젝트 개요 (Introduction)
본 시스템은 언어 교육 환경에서 수집되는 학습자의 행동 데이터(클릭, 시간, 상호작용 등)를 실시간으로 분석하여 교사의 교수 의사결정을 지원합니다. 

## 🧠 핵심 알고리즘 및 교육학적 구현
- **ZPD (근접발달영역)**: 학생 점수의 평균(mean)과 표준편차(std)를 실시간 계산하여 적정 학습 구간을 시각화합니다. 
- **SRL 지수**: Zimmerman의 모델을 수식화하여 `(시간/120 + 상호작용 + 성공)/3`으로 산출합니다. 
- **VARK 모델링**: 행동 데이터를 정규화(Normalization)하여 시각(V), 청각(A), 읽기/쓰기(R), 운동(K) 점수를 산출하고 지배적 유형을 분류합니다.
- **AI 예측**: RandomForest 이진 분류 모델을 통해 학습 성공 기여 요인을 분석합니다.
